# Time-Based Crypto Concept (NAME NEEDED TO BE UPDATED SOON)

## 1. Project Idea

We are building a time-based cryptocurrency.

1 token = 1 minute of time privilege.

The token can be used for:
- Being late to class
- Skipping time in a line

The system will simulate a small time-based economy.

# 6. Main Concept Summary

This project simulates a time-based cryptocurrency where time is treated as a scarce and tradable resource.

Tokens represent minutes of privilege and are used inside a controlled digital marketplace.

The system demonstrates:
- Basic cryptocurrency mechanics
- Supply and demand dynamics
- Transaction recording
- Secure user management
